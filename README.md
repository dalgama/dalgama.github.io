# Dilanga Algama – Portfolio

This repo contains the source for my personal portfolio site, showcasing my work as a software developer, full-stack engineer, and generative AI builder.

## Tech Stack

- **HTML5 / CSS3 / JavaScript**
- **Bootstrap 4/5** for layout and components
- Custom timelines, skills visualizations, and project cards
- Deployed as a static site (e.g., GitHub Pages / other static hosting)

## Site Sections

- **Hero** – headline, summary, and primary calls-to-action (View Projects / Contact Me)
- **About** – background and what I focus on
- **Education** – BSc in Computer Science & Management (uOttawa)
- **Experience** – uOttawa (Elentra ME, Web Services), Euroshine, etc.
- **Skills** – core languages, frameworks, and tools
- **Projects** – selected web, backend, and AI/automation projects
- **Volunteering & Rewards** – community involvement and recognition
- **Contact** – links to email, GitHub, and LinkedIn

## Local Development

```bash
git clone https://github.com/dalgama/dalgama.github.io.git
cd dalgama.github.io
# Open index.html directly in a browser OR run a simple static server:
python -m http.server 8000

```
