# Dilanga Algama - Portfolio

This repo contains the source for Dilanga Algama's personal portfolio site.
It is a fast static one-page portfolio positioning Dilanga as an Enterprise Software Engineer focused on full-stack applications, mobile app delivery, production systems, and AI-enabled platforms.

## Tech Stack

- **HTML5**
- **CSS3**
- No runtime JavaScript dependency
- Deployed as a static site through GitHub Pages or another static host

## Site Sections

- **Hero** - positioning and primary calls-to-action
- **Impact Snapshot** - delivery-focused credibility points
- **Featured Work** - enterprise, mobile, and AI-enabled software examples
- **Experience** - professional roles
- **Technical Stack** - backend, frontend, mobile, database, DevOps, and AI tools
- **Certifications** - Scrum and AI credentials
- **Contact** - email, LinkedIn, GitHub, and resume links

## Local Development

```bash
git clone https://github.com/dalgama/dalgama.github.io.git
cd dalgama.github.io
# Open index.html directly in a browser.
```

## Resume

The portfolio links to `resume.pdf` from the hero and contact sections. Add the current resume PDF at the repo root before publishing those links.
